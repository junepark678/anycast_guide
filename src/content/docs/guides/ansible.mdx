# Using Ansible to Manage Routers

One major struggle when managing your network infrastructure is keeping your routers configured consistently. Ansible, a powerful automation tool, can help you automate the configuration of your routers, ensuring that they are always up-to-date and compliant with your network policies. Ansible is a configuration tool widely used across the industry for automating IT tasks, including network device management.
